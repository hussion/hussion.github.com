linux学习笔记(六): 压缩解压缩命令

  • 命令名称： gzip和gunzip

    功能描述：压缩和解压缩文件
    ``` ruby 语法：
    gzip [文件] #压缩文件
    gzip -d [文件] #解压缩.gz格式压缩文件
    gunzip [选项] [压缩文件] #解压缩.gz格式的压缩文件

ps:
1> 压缩后的文件格式为 .gz
2> 只能压缩文件
3> 不保留原文件(直接替换原文件)

  • 命令名称： zip和unzip

    功能描述：压缩文件和目录

  • 命令名称： bzip2

    功能描述：压缩文件

  • 命令名称： bunzip2

    功能描述：解压缩文件

    PS：
    1> 如果文件不大，且对压缩比不太要求的话，推荐使用zip命令
    2> bzip2是gzip的升级版，压缩比很高!

Python

python基础知识

算法：
算法是描述如何完成一项任务的方法。实际上，在编写程序的时候，就是要使用计算机能够理解的语言（如python）来准确地描述算法。这类对机器友好的描述就叫做程序。
函数（function）：
python中的函数类似于java中的方法，可以有参数，并且返回值
模块（module）：
模块就是扩展，它可以导入到python中，从而扩展python的功能。在使用模块的时候需要先导入模块。例如： math模块提供了很多有用的数学函数。

如果不想每次都输入模块名.函数名来调用函数，想直接通过函数名来调用,可以这么做。

但是使用from…import…语句，容易造成命名冲突，比如cmath模块中也有个sqrt()方法，这样的话一旦使用此语句，就没法使用普通的sqrt函数了，所以除非真的需要from这个形式的模块导入语句，否则应该坚持使用普通的import
运算符：
对于基本的加减乘除和java都一样，对于乘除python里进行了增强。

注意：Python 不支持 C 语言中的自增 1 和自减 1 运算符,这是因为 + 和 － 也是单目运算符，Python 会将 --n 解释为-(-n) 从而得到 n , 同样 ++n 的结果也是 n.

获取用户输入：
获取用户输入可以使用input(prompt)函数和raw_input(prompt).

从以上测试，我们知道：
1. raw_input()将所有输入作为字符串看待，返回字符串类型。而input()在对待纯数字输入时具有自己的特性，它返回所输入的数字的类型（ int, float ）;
2. 这两个函数均能接收字符串 ，但raw_input()直接读取控制台的输入（任何类型的输入它都可以接收）。而对于input()，它希望能够读取一个合法的 python 表达式，即你输入字符串的时候必须使用引号将它括起来，否则它会引发一个 SyntaxError。
3. 查看Built-in Functions可以知道input()本质上还是使用raw_input()来实现的，只是调用完raw_input()之后再调用eval()函数，所以，你甚至可以将表达式作为input()的参数，并且它会计算表达式的值并返回它。 另外在Built-in Functions里说到：Consider using the raw_input() function for general input from users. 意思就是除非对input()有特别需要，否则一般情况下我们都是推荐使用raw_input()。l
